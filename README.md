# 💜 Jessica John: Portfolio

My personal developer portfolio built with React and Framer Motion. Features smooth scroll animations, a dark purple theme, and sections for my projects and skills.


🔗 **Live Site**: https://jessicajohn23.github.io/portfolio


## Features

- Animated sections using Framer Motion (fade-in on scroll)
- Sticky navbar with blur effect on scroll
- Projects section with live demo and GitHub links
- Skills grid organized by category
- Contact section with email and social links
- Fully responsive layout


## Tech Stack

- Framework: React 19 (Vite)
- Animations: Framer Motion
- Styling: CSS with custom properties (no UI library)
- Fonts: Inter + Fira Code (Google Fonts)
- Deployment: GitHub Pages


## Running Locally

```bash
# Clone the repo
git clone https://github.com/jessicajohn23/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Visit `http://localhost:5173`


## Deployment

```bash
npm run deploy
```

Builds the project and pushes to the `gh-pages` branch automatically.
